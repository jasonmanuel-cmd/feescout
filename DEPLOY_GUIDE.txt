🚀 DEPLOY IN 5 MINUTES
════════════════════════════════════════════════════════════════

STEP 1: Install Vercel CLI
────────────────────────────────────────────────────────────────
Open PowerShell and run:

  npm i -g vercel

(If you don't have npm, install Node.js from nodejs.org first)


STEP 2: Navigate to Project
────────────────────────────────────────────────────────────────
  cd output/feescout-vercel


STEP 3: Login to Vercel
────────────────────────────────────────────────────────────────
  vercel login

(Opens browser - click Continue with GitHub/Email)


STEP 4: Deploy to Production
────────────────────────────────────────────────────────────────
  vercel --prod

Follow prompts:
  ? Set up and deploy? [Y]
  ? Which scope? [Your account]
  ? Link to existing project? [N]
  ? What's your project's name? feescout
  ? In which directory is your code located? ./

Vercel will give you a URL like: https://feescout.vercel.app


STEP 5: Set Environment Variable
────────────────────────────────────────────────────────────────
1. Go to: https://vercel.com/dashboard
2. Click your "feescout" project
3. Go to: Settings → Environment Variables
4. Add new variable:

   Name:  BLOCKCHAIR_API_KEY
   Value: G___MTJ7PgczS9WHwUrUVqPVA4m4qqXf

5. Click "Save"
6. Redeploy: Run "vercel --prod" again

⚡ PERFORMANCE TIPS
════════════════════════════════════════════════════════════════

1. Enable Edge Caching
   Add to vercel.json routes:
   {"src": "/api/gas-fees/latest", "headers": {"cache-control": "s-maxage=60"}}

2. Add Redis Cache
   Store fee data for 60 seconds to reduce Blockchair API calls

3. Use Vercel KV
   Built-in Redis: vercel install kv

🆘 TROUBLESHOOTING
════════════════════════════════════════════════════════════════

Error: "Module not found: fastapi"
Fix: Make sure requirements.txt is in root folder

Error: "Deployment failed"
Fix: Check vercel.json syntax (must be valid JSON)

Error: "BLOCKCHAIR_API_KEY not set"
Fix: Add in Vercel dashboard, then redeploy

Slow response times?
Fix: Add Redis caching (see Performance Tips)
