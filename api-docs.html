Gas Fee Endpoints

GET /api/gas-fees/latest

Get the latest gas fees for all tracked blockchains. Returns fee data in USD and native token amounts. Data refreshes every 60 seconds.

Example Request (curl)
curl -s "https://feescout.bond/api/gas-fees/latest" \ -H "X-API-Key: fs_your_key_here"
Example Request (Python)
import requests resp = requests.get( "https://feescout.bond/api/gas-fees/latest", headers={"X-API-Key": "fs_your_key_here"} ) data = resp.json() for chain in data["data"]: print(f"{chain['chain']}: ${chain['fee_usd']}")
Example Request (JavaScript)
const resp = await fetch( "https://feescout.bond/api/gas-fees/latest", { headers: { "X-API-Key": "fs_your_key_here" } } ); const data = await resp.json(); data.data.forEach(chain => { console.log(`${chain.chain}: $${chain.fee_usd}`); });
Example Response
{ "success": true, "count": 41, "data": [ { "chain": "ethereum", "fee_usd": 12.50, "fee_native": 0.0042, "fee_type": "avg_usd", "timestamp": "2026-05-29T21:00:00+00:00" }, { "chain": "polygon", "fee_usd": 0.003, "fee_native": 0.0012, "fee_type": "avg_usd", "timestamp": "2026-05-29T21:00:00+00:00" } ], "updated_at": "2026-05-29T21:00:00+00:00" }
GET /api/gas-fees/cheapest

Get only the cheapest chains. Use the limit parameter to control how many results you get (default 5, max 41).

Query Parameters
# limit (optional): Number of results to return. Default: 5, Max: 41
Example Request
# curl curl -s "https://feescout.bond/api/gas-fees/cheapest?limit=3" \ -H "X-API-Key: fs_your_key_here" # Python import requests resp = requests.get( "https://feescout.bond/api/gas-fees/cheapest", params={"limit": 3}, headers={"X-API-Key": "fs_your_key"} ) # JavaScript const resp = await fetch( "https://feescout.bond/api/gas-fees/cheapest?limit=3", { headers: { "X-API-Key": "fs_your_key" } } );
GET /api/gas-fees/comparison

Get a side-by-side comparison of fees across all chains with savings data vs Ethereum. Perfect for dashboards and decision-making tools.

Example Request
curl -s "https://feescout.bond/api/gas-fees/comparison" \ -H "X-API-Key: fs_your_key_here"

Health check endpoint. Returns API status, version, and uptime info. No authentication required.
